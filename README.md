# Nextcloud Bot

Various nextcloud automations.

All configured through collectives page in Nextcloud.

Features:

- Calendar appointment reminder
- Mailinglist
- Membership overview, who contributes where
- Deck reminder
- Protocol summaries
- Logbook overview
